Contacts Server

REST API documentation for the Contacts HTTP server (used in thoughtram AMC).

  • getRoutes() /api/routes GET
  • getRoute(id) /api/routes/:id GET
  • searchRoutes(query) /api/search?text==<xxx> GET
  • checkEmail(email) /api/check-email?email=<xxx> GET
  • createRoute(route) /api/routes/:id PUT
  • updateRoute(contact) /api/route PUT
  • deleteRoute(id) /api/routes/:id DELETE
